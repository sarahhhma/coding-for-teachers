Coding for Teachers!

I know it seems scary, but you can do it! Look how quickly your students pick up this skill. Pick one lesson at a time and work through it with your students or after school. In no time you will be comfortable teaching basic coding!

Block Coding

Block coding doesn't have to be for kids! This style of coding makes it easier to make connections between the functions of different bits of code and how they fit together. Use this as a good way to get yourself acquainted with programming. You will be ready for syntax coding before you know it!

Syntax Coding

If you are ready for a bit of a challenge, you can dive into syntax coding. What's important to note here is that the differences between different coding languages are very important to note. Skill Struck mostly sticks to HTML in elementary school, which has a very formulaic syntax, making it a good starting point for programming.
